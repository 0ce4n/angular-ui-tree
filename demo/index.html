NestedSortable demo's

NestedSortable is an AngularJS component that can render nested lists wit drag and drop support.

GitHub repository

Sample 1

A nested list, it binds with a nested array, if it's sort order is changed, the data will be updated. In this sample, it can only support 3-level nodes.

Try sample 1

Sample 2

Two-level list, the data type of top level is 'chapter', the data type of the secend level is 'lecture', the node can only be dragged & dropped to another node which with same data type.

Try sample 3

Sample 3

A nested recursive list, it binds with an unlimited nested array.
